# SSH Reverse Proxy Tunnel Manager

Приложение для Windows с графическим интерфейсом и значком в трее для управления Reverse Proxy SSH Tunnel-подключениями.

## Возможности
- Добавление, удаление и управление SSH Reverse Proxy Tunnel-подключениями
- Хранение подключений в tunnel.json (пароли зашифрованы)
- Автоматическое подключение активных туннелей при запуске
- Значок в трее с быстрым доступом к настройкам и управлению
- Автозагрузка при старте Windows

## Установка и запуск из исходников

1. Установите Python 3.8+
2. Установите зависимости:
   
   pip install -r requirements.txt

3. Запустите приложение:
   
   python main.py

## Зависимости
- PyQt5
- paramiko
- cryptography

## Инструкция
- Для добавления нового подключения нажмите [+] в интерфейсе.
- Для активации/деактивации используйте тумблер в таблице.
- Для удаления используйте кнопку удаления.
- Все активные подключения автоматически запускаются при старте программы.
- Для выхода или отключения автозагрузки используйте меню в трее.

## Автозагрузка
Программа добавляет себя в автозагрузку Windows через реестр. Для удаления автозагрузки используйте пункт меню "Отключить" в трее.

## tunnel.json
Файл tunnel.json создается рядом с программой и содержит список подключений. Пароли хранятся в зашифрованном виде.
